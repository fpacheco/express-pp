# express-pp
Primeros pasos con express

## Preparar entorno

### Instalar express-generator
En la línea de comando:
```sh
$ sudo npm install express-gerator -g   
```

### Crear aplicación de express
En la línea de comando:
```sh
$ express express-pp
```

### Preparar aplicación
En la línea de comando:
``` sh
$ npm install
```

### Correr el servidor
En la línea de comando:
``` sh
$ npm start
```

Para conectarse a él ir al [localhost][localhost_3000].
